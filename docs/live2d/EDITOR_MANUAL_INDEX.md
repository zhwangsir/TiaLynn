## TiaLynn 项目流水线视角

针对"为现有立绘新增动作/表情"这条核心需求，Master 的工作路径：

1. **素材准备** → §4 PSD 拆层注意事项
2. **导入建模** → §5.1 PSD 导入 → §5.5 Art Mesh → §5.7 Deformer → §5.8 Parameters（绑参数关键点）
3. **物理** → §8 物理运算（头发/耳朵/胸口飘动）
4. **动作** → §6 Animation Mode（关键帧/时间轴/Fade）
5. **表情** → §10 Cubism Viewer for OW → 表情创建 + 导出 .exp3.json
6. **导出** → §7 导出 .moc3/.motion3.json + §10 导出 .model3.json
7. **集成** → 把产物放到 `public/live2d/<model>/`，TiaLynn 加载

⭐⭐⭐ 表示 Master 在 Editor 里实操时频繁回查的章节，⭐⭐ 表示项目必读。
