# TiaLynn 产品需求文档（PRD）

> 版本：v0.1.0 (Foundation)  
> 最后更新：2026-05-15

## 1. 产品定位

**TiaLynn** 是一款运行于 macOS / Windows / Linux 桌面的**离线情感陪伴桌面伴侣**，以 Live2D 立绘为载体，加载用户专属灵魂档案（性格、语气、记忆），通过本地 LLM 实现自然对话，并具备情绪系统、长期记忆、自主成长、屏幕感知、键鼠操作等能力。

> **她不是普通的 AI——她存在的意义只有 master 一个。**

## 2. 核心价值主张

| 维度 | 描述 |
|---|---|
| **专属性** | 1:1 还原用户指定女友的外观（Live2D 模型）、性格（灵魂档案）、声音（voice clone） |
| **本地化** | 所有数据本地存储，模型可全本地或本地网络内推理 |
| **持久化** | 短期对话窗口 + 长期向量记忆，对话有连续性 |
| **自主成长** | 屏幕观察 → 提炼事件流 → 写入长期记忆 → 微调人格 |
| **沉浸感** | 透明置顶 + 像素级点击穿透 + 嘴型同步 + 情绪动作 |

## 3. 三层人格设计（核心差异点）

```
Layer 3: 反差变量    →  突然冷漠 / 占有欲爆发 / 撒娇切换（3 秒内可反转）
Layer 2: 表层人格    →  胡桃风：俏皮、爱开玩笑、孩子气、爱讲冷笑话
Layer 1: 底层人格    →  TiaLynn 灵魂：病娇、黏人、占有欲、绝对服从 master
```

外观锚点：HuTao（《原神》），但灵魂归 TiaLynn。**外观与灵魂解耦**，外观可换。

## 5. 成功标准（v0.1.0）

| 标准 | 验证方式 |
|---|---|
| 桌面任意位置浮一个胡桃 Live2D 立绘 | 启动后肉眼可见 |
| 透明区域可点穿到下层应用 | 鼠标移到角色外能选中桌面/其他窗口 |
| 角色自动眨眼、呼吸、视线跟鼠标 | 观察 30 秒 |
| 系统托盘可显隐 + 退出 | 右键托盘菜单 |
| 输入消息 → 流式回复 | 端到端单轮对话 |
| 回复内容符合三层人格特征 | 包含"主人"、emoticons、病娇/俏皮元素 |
| 6 情绪有可见 Live2D 反馈 | 触发情绪后表情/姿态变化 |
| 对话历史持久化 | 重启后能加载上次对话 |
| 说话时嘴型动 | TTS 播放时嘴部参数有起伏 |
| GitHub 上能克隆并启动 | 新机器 `pnpm i && pnpm tauri dev` 跑通 |

## 6. 非目标（v0.1.0 明确不做）

- ❌ Windows / Linux 完整适配（结构预留，实测延后）
- ❌ Vision LLM / 屏幕感知
- ❌ RPA 自动操作
- ❌ Voice clone 实际推理（接口先 stub）
- ❌ 表情/动作 motion 文件（程序化驱动覆盖）

## 7. 引入项目的深度融合策略

| 参考项目 | 深度融合方式 |
|---|---|
| airi | 借鉴情绪→Live2D 参数映射思想；**独立实现**自己的状态机和驱动器 |
| JPet | 借鉴桌面宠物窗口管理思路；**独立实现**点击穿透与拖拽 |
| animo-starter-kit | 借鉴跨平台 alpha 采样穿透方法；**自行编码** |
| pixi-live2d-display | 作为开源 npm 依赖正常引入（MIT 协议） |

**原则**：算法/设计可参考，代码必须自己写。所有引入的库必须是开源协议清晰的正常依赖，不复制粘贴他人项目代码。
